# 🎬 Movie Genre Classification

## 📌 Project Overview
This project is developed as part of the **CodSoft Machine Learning Internship**.

The goal of this project is to build a Machine Learning model that predicts the genre of a movie based on its plot summary or description. The model learns patterns from historical movie data and classifies unseen movie descriptions into their most likely genres.

---

## 🎯 Objective

- Predict movie genres using Machine Learning.
- Perform text preprocessing on movie descriptions.
- Convert text into numerical features using TF-IDF Vectorization.
- Train and evaluate a classification model.
- Predict genres for new movie descriptions.

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- TF-IDF Vectorizer
- Label Encoder

## 📊 Dataset

The dataset contains movie information including:

- Movie Title
- Genre
- Movie Description

The dataset is used to train and test the genre classification model.

---

## ⚙️ Workflow

1. Load Dataset
2. Data Cleaning
3. Text Preprocessing
4. Label Encoding
5. TF-IDF Vectorization
6. Train-Test Split
7. Model Training
8. Model Evaluation
9. Save Trained Model
10. Predict Genre for New Movies
